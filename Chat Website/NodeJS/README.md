##### NodeJS

###### Instalare NodeJS
https://nodejs.org/en/

###### Instalare Dependinte
Navigati in directorul NodeJS (unde se afla package.json)
Deschideti cmd in acest director si rulati:

npm install

###### Rulati server-ul
Fisierul principal al acestui server este fisierul index.js. Asa ca rulam comanda:

node index.js
